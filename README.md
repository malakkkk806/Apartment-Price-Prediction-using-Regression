# Apartment-Price-Prediction-using-Regression
This project applies machine learning regression techniques to predict apartment prices based on real estate listing features such as location, property type, size, number of bedrooms, bathrooms, availability status, and payment method.

The notebook includes data cleaning, preprocessing, feature transformation, model training, and evaluation using regression metrics such as MAE, RMSE, and R². Multiple regression models are compared to identify the best-performing approach for estimating apartment prices.
